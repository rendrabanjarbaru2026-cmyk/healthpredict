AWS Configuration Notes

This frontend is hosted on AWS Amplify. The API Endpoint and API Key are injected as environment variables (API_ENDPOINT, API_KEY) in the Amplify Console under App settings → Environment variables. Configure those variables in Amplify and redeploy — the values will be used automatically by the build process and embedded in this page's initialization script.
